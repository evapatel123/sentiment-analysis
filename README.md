# 🧪 Sentiment Analysis Lab 
First, make a new copy of this notebook on Google Colab. Next, go to `Edit` and select `Clear all outputs`. This will make sure that when you run your code, the output is not already shown.

This lab is about showing one way to analyze sentiment in text messages using Python. This includes cleaning and processing messages, counting word frequencies to extract meaningful features, and then use these features to train and test a logistic regression model that classifies messages as positive or negative. This helps in understanding the entire pipeline—from data preparation to model evaluation. Here are the steps needed to complete the full process:

1️⃣ Load and clean a dataset of messages with sentiment labels.

2️⃣ Build a word frequency dictionary to count word appearances.

3️⃣ Visualize common words to see which words strongly indicate sentiment.

4️⃣ Extract features from messages to represent each message as a vector.

5️⃣ Train and test a logistic regression model!
